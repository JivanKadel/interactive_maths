# Cardinality of Sets

The **cardinality** of a set is the number of elements in it.

- If `A = {1, 2, 3, 4}`, then `n(A) = 4`
- Finite set: has countable elements
- Infinite set: like natural numbers `N = {1,2,3,...}`
