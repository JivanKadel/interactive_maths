# Cardinality Examples

**Example 1:**  
`A = {x | x is a vowel in English}`

- `A = {a, e, i, o, u}` → `n(A) = 5`

**Example 2:**  
`B = {x | x is a prime number < 20}`

- `B = {2,3,5,7,11,13,17,19}` → `n(B) = 8`
