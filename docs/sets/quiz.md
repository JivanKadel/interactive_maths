# Sets Quiz

1. If `A = {1,2,3}` and `B = {2,3,4}`, what is A ∩ B?
2. Which of the following is a finite set?
   - `N = {1,2,3,...}`
   - `{a,b,c}`
   - The set of all stars in the universe
3. Let `U = {1..10}`, `A = {2,4,6,8}`, find Aᶜ.
