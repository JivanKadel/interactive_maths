# Mathematics – Concept Quiz

## Sets

### Short Questions

1. Define a finite and infinite set with examples.

2. If $A = \{1, 2, 3, 5, 6, 7\}$ and $B = \{3, 4, 5\}$, find $A \cup B$ and $A \cap B$. Is B a subset of A?

### Long Question

- Given $U = \{1, 2, 3, 4, 5, 6, 7, 8\}$, $A = \{2, 4, 6\}$, $B = \{4, 5, 6\}$
  Find:

  a) $A \cup B$  
  b) $A \cap B$  
  c) $A - B$  
  d) $B'$ (complement of B)

## Trigonometry

### Formulas

- $ \sin \theta = \dfrac{\text{Opposite}}{\text{Hypotenuse} } $

- $ \cos \theta = \dfrac{\text{Adjacent}}{\text{Hypotenuse} } $

- $ \tan \theta = \dfrac{\text{Opposite}}{\text{Adjacent} } $

### Short Questions

1. Define sine and cosine ratios.

2. If $\sin \theta = \dfrac{3}{5}$, find $\cos \theta$.

### Long Question

- In a right triangle, if one angle is $30^\circ$ and hypotenuse is 10 cm, find the opposite and adjacent sides using trigonometric ratios.

    <li>C) 1/2</li>
    <li>D) 2/5</li>

    <summary>Reveal Solution</summary>

  </details>

# Conceptual Practice – Grade 9 Math

---

## Trigonometry

1. If $ \sin \theta = \dfrac{3}{5} $ and the hypotenuse is 20 cm, find the base.

2. In a right triangle, $ \tan \theta = \dfrac{5}{12} $. Find $ \cos \theta $.

3. If $ \cos \theta = \dfrac{4}{5} $, find $ \tan \theta $.

4. A ladder leans against a wall making an angle of $60^\circ$ with the ground. If the ladder is 10 m long, find the height it reaches on the wall.

---

## Sets

1. Let $A = \{x: x \text{ is a multiple of 3, } x < 20\}$ and $B = \{x: x \text{ is even, } x < 20\}$. Find $A \cap B$.

2. If $U = \{1,2,3,4,5,6,7,8\}$, $A = \{2,4,6\}$, $B = \{4,5,6\}$, find $A \cup B$ and $A \cap B$.

3. True or False: $A \subseteq A \cup B$ for any sets $A$ and $B$.

4. If $n(A) = 10$, $n(B) = 15$, and $n(A \cap B) = 5$, find $n(A \cup B)$.

---

## Triangle

1. In triangle $ABC$, if $\angle A = 2x$, $\angle B = 3x$, and $\angle C = 5x$, find the value of $x$.

2. A triangle has sides 5 cm, 12 cm, and 13 cm. Is it a right triangle?

3. The perimeter of a triangle is 36 cm. Two sides are 12 cm and 15 cm. Find the third side.

---

## Statistics

1. The marks obtained by a student in 5 subjects are: 60, 70, 80, 90, 100. If the highest mark is removed, how does the mean change?

3. A data set has a mean of 20. If one value 30 is replaced by 10, what happens to the mean?

4. The mode of a data set is 8. If one more 8 is added, does the mode change?
