Q1. In triangle ABCABCABC, if ∠A=50∘\angle A = 50^\circ∠A=50∘ and ∠B=65∘\angle B = 65^\circ∠B=65∘, find ∠C\angle C∠C.

Show solution

Solution: ∠C=180∘−(50∘+65∘)=65∘\angle C = 180^\circ - (50^\circ + 65^\circ) = 65^\circ∠C=180∘−(50∘+65∘)=65∘
